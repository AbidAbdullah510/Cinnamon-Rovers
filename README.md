# Cinnamon Rover Project Website

Welcome to the **Cinnamon Rover Project Website**, a futuristic and interactive multi-page site built to showcase my autonomous rover project. This website features dashboards, control panels, project data, achievements, and creator information, all with neon-themed animations and dynamic effects.

---

## **Website Features**

### Home Page
- Full-width background video  
- Neon futuristic theme  
- Rover introduction and description  

### Achievements Page
- Placeholder for future achievements  
- Animated neon text and cards  

### Control Panel Page
- Interactive push buttons and sliders  
- Real-time button animations  
- Ready to send values to Arduino via Web Serial API  

### Data Display Page
- Sensor data dashboard (temperature, humidity, soil moisture, harmful gas)  
- Animated gradient cards and floating effects  

### Settings Page
- Mechanical/metal-inspired theme with CSS animated gears  
- Placeholder text: *"No settings feature available yet"*  

### About Creator Page
- Typewriter effect for name, hard skills, and soft skills  
- Floating project cards with hover and scroll animations  
- Responsive neon midnight blue theme  

### Picture and About Page
- Detailed rover information  
- Project images and descriptions  

---

## **Technologies Used**

- HTML5, CSS3, JavaScript (ES6)  
- CSS Animations and Transitions  
- Background videos  
- Neon & futuristic design  
- Web Serial API for Arduino integration (future)
